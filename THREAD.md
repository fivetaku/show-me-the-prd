# 스레드 메인글

---

바이브코딩을 시작한 사람들이
가장 어려워하는 건??

**개발 문서 만들기.**

PRD, 기능명세서, ERD, 기술 스펙...
개발 안 하는 사람은 이게 뭔 말인지도 모르고
뭘 작성해야 하는지도 모릅니다.

그냥 "인보이싱 앱 만들어줘" 하면
AI가 알아서 해주겠지? → 결과 엉망.

---

그래서 만들었습니다.

PRD 치트키, **쇼미더피알디** 🎤

한 문장만 말하면 끝.
AI가 알아서 질문하고, 알아서 조사하고,
기획 문서 4종을 자동으로 만들어줍니다.

```
/show-me-the-prd 사진 정리 앱 만들고 싶어
```

---

**뭘 해주냐면:**

5-6번 질문에 답하면 (모르겠으면 "추천" 고르면 됨)

📋 PRD -- 뭘 만드는지, 뭘 안 만드는지
📊 데이터 모델 -- 앱의 핵심 데이터 구조
📅 Phase 분리 -- 단계별 개발 계획 + 시작 프롬프트
⚙️ 프로젝트 스펙 -- AI한테 "이건 절대 하지 마" 규칙

기술 스택도 실시간 웹 리서치 해서 추천해줍니다.
기능별 복잡도도 알려줘서 첫 버전에 뭘 넣고 뺄지 판단 가능.

---

**설치 (Claude Code 전용, 무료)**

```
/plugin marketplace add https://github.com/fivetaku/gptaku_plugins.git
/plugin install show-me-the-prd
```

GitHub: https://github.com/fivetaku/show-me-the-prd

AI한테 "만들어줘" 하기 전에
"뭘 만들 건지"부터 정리해보세요.
